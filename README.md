## ⌨️ Copilot Shortcuts

<div align="center">

| Shortcut    | Mode   | Action              |
| ----------- | ------ | ------------------- |
| `<M-Tab>`   | Insert | Accept suggestion   |
| `<M-]>`     | Insert | Next suggestion     |
| `<M-[>`     | Insert | Previous suggestion |
| `<C-]>`     | Insert | Dismiss suggestion  |
| `<M-\>`     | Insert | Trigger completion  |
| `<M-Right>` | Insert | Accept next word    |



</div>

## 🚀 Quick Installation

```bash
# Clone the repository into Neovim configuration directory
git clone https://github.com/VictorzllDev/nvim.git ~/.config/nvim

# Start Neovim to automatically install plugins
nvim
```
